# Cyberise Technology Site Stack

## Frontend

| Layer | Technology |
|---|---|
| Framework | React 19 |
| Language | TypeScript |
| Build tool | Vite 7 |
| Styling | Tailwind CSS v4 |
| Routing | wouter |
| 3D / animation | Three.js |
| Motion | Framer Motion |
| Component library | Radix UI |
| Form handling | React Hook Form + Zod |
| Icons | Lucide React + React Icons |
| Fonts | Google Fonts (Inter, Orbitron) |

## Backend

| Layer | Technology |
|---|---| 
| Runtime | Node.js 24 |
| Framework | Express 5 |
| Serverless functions | Vercel Functions |
| Email delivery | Resend |
| Logging | Pino + pino-http |
| Validation | Manual validation + regex email check |

## Development Environment

| Layer | Technology |
|---|---|
| Workspace | Replit |
| Monorepo tool | pnpm workspaces |
| Package manager | pnpm 10 |
| TypeScript | 5.9 |
| Code quality | Prettier |

## Deployment

| Layer | Technology |
|---|---|
| Platform | Vercel |
| Frontend hosting | Static hosting |
| API hosting | Vercel Serverless Functions |
| Routing | vercel.json rewrites |
| CI/CD | GitHub auto-deploy |

## Email Infrastructure

| Layer | Technology |
|---|---|
| Provider | Resend |
| Sender domain | contacts@cyberise.org |
| Recipient | Cyberisetecnologies@consultant.com |

## Summary

This site is a React single-page application built with Vite, styled with Tailwind, deployed on Vercel, and powered by a serverless contact form using Resend.
